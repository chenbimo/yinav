## 这是什么？

这是随易科技旗下开源的一个 `网址导航` 项目，提供了强力的后端接口，完善的后台管理和精美的导航网站。

你可以使用本项目，很方便快捷地部署到自己的服务器上，获得属于自己的 `网址导航`。

## 功能计划

**v1**

1. [ ] 一个网址有多个导航，聚合展示。
2. [ ] 导航可以切换列表和卡片模式。
3. [ ] 增加显示导航的图标。
4. [ ] 导航可以排序。
5. [ ] 后台管理显示分类的名称。
6. [ ] 检测网址的有效性（优先级低）。

**v2**

1. [ ] 增加导航首页功能，选取部分导航组成首页。
